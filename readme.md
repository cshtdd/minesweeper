# Console Minesweeper  

Console implementation of a basic Minesweeper game with Java and TDD.  

[Minesweeper Spec](https://en.wikipedia.org/wiki/Minesweeper_(video_game))  

## Run the application  

```bash
java -cp target/minesweeper-1.0.jar Program
```
